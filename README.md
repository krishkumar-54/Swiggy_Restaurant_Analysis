# Swiggy Restaurant Data Analysis

## Project Overview

This project analyzes Swiggy restaurant data to uncover insights into restaurant performance, customer engagement, pricing patterns, and delivery efficiency across multiple cities in India.

The project demonstrates an end-to-end data analytics workflow using Python, SQL, and Power BI.

---

## Objectives

* Analyze restaurant distribution across cities.
* Identify top-rated and most popular restaurants.
* Evaluate delivery performance by city.
* Categorize restaurants based on pricing.
* Explore customer engagement through rating counts.
* Build an interactive Power BI dashboard for business insights.

## Key Insights

* Chennai has the highest average restaurant rating.
* Kolkata has the largest restaurant presence.
* Mid-Range restaurants account for the largest share of listings.
* Most restaurants fall under the Moderate delivery category.
* Delivery time shows a weak negative correlation with ratings.
* Customer engagement varies significantly across restaurants.
